Basic Concepts

Web 1.0, 2.0 and 3.0 are conceptual buzz words ment to describe the everchanging World-Wide-Web.

Web 1.0:
Also known as the “readable” phrase of the World Wide Web with flat data. In Web 1.0, there is only limited interaction between sites and web users. Web 1.0 is simply an information portal where users passively receive information without being given the opportunity to post reviews, comments, and feedback.

Web 2.0:
Also known as the “writable” phrase of the World Wide Web with interactive data. Unlike Web 1.0, Web 2.0 facilitates interaction between web users and sites, so it allows users to interact more freely with each other. Web 2.0 encourages participation, collaboration, and information sharing.
Some websites considered web 2.0 cannot be existed without users which are contributers, for example: Facebook would vanish if users will not contribute themselfs to it.

Web 3.0:
Also known as the “executable” phrase of Word Wide Web with dynamic applications, interactive services, and “machine-to-machine” interaction. Web 3.0 is a semantic web which refers to the future. In Web 3.0, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of users. One example of Web 3.0 is Tivo, a digital video recorder. Its recording program can search the web and read what it finds to you based on your preferences.

Q1: "What are 3 differences between those webs?"
A1: Let's see it in the following table

Web 1.0 Web 2.0 Web 3.0
Usages Mainly for information portals where users passively receive information without being given the opportunity to post reviews, comments, and feedback. Mainly websites which incourage participation, collaboration, and information sharing between users. That also include all peer-to-peer eCommerce websites (eBay for example). Mainly for indroducing state of the art technologies into the website user experience. These sites use AI and IoT to make the web feel like an intelligent human is out there making sure you will get what you need.
Site Contributers Only the website owner/handler is adding information. Users also add infomation to the web by commenting, reviewing and posting. Machines also can add infomation into the web, for example your smart watch can post your current heart rate to your site.
Used Technologies HTML, CSS on Client side and Perl, PHP, Python, Ruby, as well as Enterprise Java (J2EE) and Microsoft.NET Framework on Server side Introduced AJAX, JavaScript, Adobe Flash to make the websites dynamic Semantic Web, Artificial Intelligence, 3D Graphics, Connectivity, Ubiquity to make the websites much 'smarter'

Q2: "Is there any difference between the webs if im shopping for a new computer?"
A2: Sure there is, lets see

In Web 1.0 shopping for a computer would force you to enter to every website that you know that sells computers and look for the computer that you like. If you are lucky you would find a shopping portal that would gather all shops of computer for you.
In Web 2.0 websites such as eBay have come to life. This means that now you surely have a portal of shops selling computers, and now also regular people (not stores) can post their own used computer or a brand new one. This would give you a much better choice and information about whats out there to buy.
In Web 3.0 you can enter a website such as Amazon and list all the requirments that you wish on the computer. The website would use his own algorithms to match you with the best product he can find. Also the website can recommend you of computers which are common by your internet profile as well as give you a completing products for the one you looked for - this would make your life much simpler!
